# next-lets

Strona **Let's automate** — Studio transformacji AI. Odtworzenie designu (Claude Design `Strona Let's automate v9`) jako responsywna, full-bleed strona w [Astro](https://astro.build).

## Rozwój

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # build produkcyjny do ./dist
npm run preview  # podgląd builda
```

## Stack

- Astro (static output)
- Czysty HTML/CSS + odrobina vanilla JS (rotator fraz, licznik rachunku, zakładki oferty, skalowana makieta hero)
- Font: Schibsted Grotesk
